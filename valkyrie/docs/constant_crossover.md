# Constant Comparator

Allows comparisons to be performed between ticks of a node and a constant value. The output is a binary 1 (_true_) or 0 (_false_). Supported comparisons are:

* Greater Than
* Greater Than or Equal
* Less Than
* Less Than or Equal
* Equal
* Not Equal


### Formula

$A = \alpha \ @ \ V$

$\alpha$ = Input 1

$V$ = Constant Value

$@$ = Comparator

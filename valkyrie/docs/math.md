# Math (MATH)

Support for basic math to be performed between ticks of two nodes. [Learn More](https://en.wikipedia.org/wiki/Partial_differential_equation)

### Supported Operations

* Addition
* Subtraction
* Multiplication
* Division

## Formula

$MATH = \alpha \ ? \ \beta$

$\alpha$ = Input 1

$\beta$ = Input 2

$?$ = Operation performed

# Derivative (DER)

Takes the derivative of line in respect to the period length.

_Default Period = 1_

## Formula

$DER = \alpha_0 - \alpha_n$

* $\alpha_0$ = currrent value

* $\alpha_n$ = value $n$ ticks ago

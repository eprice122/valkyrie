# Node Comparator

Allows comparisons to be performed between ticks of two nodes. The output is a binary 1 (_true_) or 0 (_false_). Supported comparisons are:

* Greater Than
* Greater Than or Equal
* Less Than
* Less Than or Equal
* Equal
* Not Equal


### Formula

$A = \alpha \ @ \ \beta$

$\alpha$ = Input 1

$\beta$ = Input 2

$@$ = Comparator
